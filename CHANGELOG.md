# Changelog

All notable changes to the "Solar Theme" extension will be documented in this file.

## [0.1.0] - 2026-01-10

### Added
- Initial release
- Automatic theme switching based on sunrise/sunset
- Support for day and night themes
- Optional dawn and dusk transition themes
- Status bar indicator showing current sun phase
- Commands: Update Now, Show Sun Times, Toggle Enabled
- Configurable update interval
- Optional notifications when theme changes
- Location-based sun time calculations using SunCalc
